# Atlas

Atlas is an open-source robotics library focused on odometry, motion control and autonomous navigation for competitive robotics.

## Planned Features

- [ ] 2 Wheel Odometry
- [ ] 3 Wheel Odometry
- [ ] Inertial Integration
- [ ] PID Controller
- [ ] Drive To Point
- [ ] Turn To Angle
- [ ] Pure Pursuit
- [ ] Path Planning

## Supported Platforms

- VEX V5
- FRC
- Generic Java Projects
